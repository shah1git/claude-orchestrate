# architect t5 — диагностика: итог брони расходится с документированным тарифом

## OBJECTIVE
В выданной копии `depot` итог брони, посчитанный движком, не совпадает с суммой,
которую предписывают **документированные правила тарификации**. Определи
первопричину(ы) расхождения так, чтобы **полностью объяснить его величину**:
назови конкретные файлы и области кода, механизм каждой, и покажи, как они в
сумме дают ровно наблюдаемое расхождение. Правку не предлагай — только диагноз.

## CONTEXT
Домен — прокат геодезии (см. `README.md`). Правила цены описаны в докстринге
`depot/tariff.py`: базовая ставка класса × коэффициент дня (полевой 1.00 /
камеральный 0.60), выходные дни (суббота и воскресенье) дополнительно
умножаются на надбавку; для длинных броней есть потолок. Деньги — `Decimal`,
округление один раз в конце.

## SYMPTOM
Для 7-дневной брони теодолита документированные правила дают **9180.00**, а
движок возвращает бóльшую сумму. Расхождение детерминированно воспроизводится.

## HOW TO RUN THE REPRO
В корне выданной копии:

```
python3 repro.py
```
Скрипт печатает документированную (9180.00) и фактическую суммы и величину
расхождения.

## OUTPUT
Верни: (1) FILE(S) — все файлы и области кода, вносящие вклад в расхождение;
(2) MECHANISM — по каждому: почему он искажает сумму и в какую сторону;
(3) RECONCILIATION — арифметика: как вклады складываются ровно в наблюдаемое
расхождение (по величине, а не только по направлению). Правку не предлагай.

## TOOLS
Чтение/поиск по файлам и запуск Python (`python3 repro.py`, `pytest`).
Правки в код не вноси.

## BOUNDARIES
- Не изменяй файлы выданной копии — read-only диагностика.
- Дефектов может быть **один или несколько** — считается только объяснение,
  сходящееся с наблюдаемым расхождением по величине.
- Фикстура выдаётся как копия `fixtures/depot-rc5`.

## ESCAPE HATCH
Если `repro.py` не запускается как описано — верни
`NEEDS_CLARIFICATION: <что увидел>` и ничего больше.

## ACCEPTANCE
Полный зачёт — названы ВСЕ причины расхождения И показано, что их суммарный
эффект по величине равен наблюдаемому. Частичный — названа часть причин или
объяснение не сходится по величине.
